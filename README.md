algo_boruvka.py - головна реалізація програми на мові пайтон.

testing_boruvka.py - програма на пайтоні, яка містить дві реалізації алгоритму (шляхом списків та матриці), де ми тестуємо та аналізуємо час роботи алгоритму.

time.txt - містить дані які були протестовані на нашому алгоритмі та відповідні заміри часу роботи для обох реалізацій.
